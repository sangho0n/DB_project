// 프로젝트 디렉토리 구조 관련
웹페이지 구현에 필요한 프론트엔드 파일은 모두 webContent 폴더 내부에 위치해야합니다.
기능별 구분을 위해 css 파일 styles폴더 내부에 저장해주시면 감사하겠고, UI 디자인에 필요한 이미지 파일들은 하위디렉토리인 images 폴더 안에 두었습니다.

java_src 폴더 안에는 백엔드 구현에 필요한 빈즈 파일 등의 java파일을 저장하는 폴더로 사용하면 됩니다.

UIdesign폴더 안에는 제가 디자인해본 UI와 그에 대한 설명 파일들이 들어있습니다. 이를 참고하여 html css javascript 혹은 jsp코드를 짜면 될 것 같습니다.

// 데이터베이스 관련
mysql을 이용해 DB 구축은 완료하였으나, 외부 컴퓨터에서 제 컴퓨터 DB로 접근할 수 있도록 하는 설정을 알아봤는데
어떤 방법을 동원해도 접속이 되질 않습니다.. 부득이하게 json파일을 보내드릴테니 해당 파일들을 이용해 로컬 서버에서 DB를 따로 구축하신 후, 해당 db를 이용해야 할 것 같습니다.
json 파일들은 DB 폴더 안에 있습니다.

그리고 저희 보고서에서 나타낸 데이터베이스 모델로는 각 구별 총 도로의 길이를 나타내는 정보는 포함할 수 없어가지고, gross_lengths 릴레이션을 하나 새로 만들었습니다.
최종보고서 제출 시 RDB 및 ER 모델을 수정해서 반영해야 할 것 같습니다.
